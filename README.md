## 🚀 Featured Projects

### 🏗️ [BIM Data Pipeline](https://github.com/irenele01/Data_Pipeline_Project)
> **Automated AI classification for construction elements (NBS Uniclass)**

A fully containerised end-to-end pipeline that ingests IFC files, classifies BIM elements using a hybrid RAG + LLM system, and routes results through a human-in-the-loop review queue — cutting classification turnaround from **days to minutes**.

`Apache Airflow` `Docker` `PostgreSQL` `RAG` `Groq API` `IfcOpenShell` `Python`

- 🎯 **92%+ confidence** on auto-accepted predictions
- ⚡ **Bronze → Silver → Gold** three-layer data architecture
- 👥 Human-in-the-loop review with full audit trail and state lifecycle management

---

### 🔥 [Early Fire Detection System (EFDS-Lite)](https://github.com/irenele01/Early-Fire-Detection)
> **Real-time fire & smoke detection on a Raspberry Pi 3 — $0 to deploy**

A privacy-first edge AI system that watches your server room 24/7 using a repurposed iPhone as a camera. Detects fire and smoke visually (and optionally thermally), trained on 40,000+ labelled images, with Telegram alerts in under 2 seconds.

`YOLOv8` `ONNX Runtime` `OpenCV` `Python` `Streamlit` `SQLite`

- ⚡ **Sub-500ms inference** on constrained edge hardware
- 🛡️ **<1% false positive rate** via 5-frame persistence filtering
- 🔄 **Active learning loop** — false positives feed back into retraining
